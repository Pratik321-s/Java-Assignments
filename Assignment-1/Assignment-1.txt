=> Java is a Programming Language used to build a applications.

2. Is Java platform dependent or independent?
=> Platform Independent.

3. What is JVM?
=> It is Java Virtual Machine that runs the Java programs.
 
4. What is JDK and JRE?
=> Java Development Kit is used to develop Java programs, and Java Runtime Environment used to run them. 

5. Why is Java called “Write Once, Run Anywhere”?
=> Because Java code run on any system that has JVM installed.

=>  A keyword is a reserved word with a special meaning in Java.
 
19. Can we use reserved keywords as variable names?
=>  No, reserved keywords cannot be used as a variable names.
 
20. What is the difference between = and ==?
=>  = used to assigning a value and == used for comparison.

21. What is an if statement?
=>  if statement is used to check condition and execute if the condition is true.
 
22. Difference between if and switch?
=>  if is used for multiple conditions and switch is used for checking one variable against many values.

23. What is a loop?
=>  A loop is used to execute a block of code repeatedly.
 
24. Types of loops in Java?
=>  for loop, while loop and do-while loop. 
 
25. Difference between for loop and while loop?
=>  for loop used when we know the number of iterations and while loop is used when the number of iterations are unknown.
 
26. What is break statement?
=>  break is used to stop the loop immediately.
 
27. What is continue statement?
=>  continue is used to skip the current iteration and continue the loop.
